# Node RESTfull API training
I made this project for training REST API with nodejs and express ORM.

## Also in this project:
- Sucrase
- ESlint
- Prettier
- Nodemon
- Sequelize-cli and Sequelize
- bcrypt
- JWT (JsonWebToken)

## Installation and Usage

Do as bellow:

```
yarn

yarn dev

```
